# TO Best — توثيق شامل للمشروع

## نظرة عامة

**TO Best** هو تطبيق Flutter احترافي لإدارة التدريب الرياضي والتغذية، مبني على معمارية Riverpod مع Google Apps Script كـ backend.

---

## الفكرة العامة

يوفر التطبيق نظاماً متكاملاً بين المدربين والمتدربين يشمل:
- **تتبع التمارين** مع سجل مفصل لكل جلسة
- **تغذية ذكية** مع قاعدة بيانات غذائية بالعربية والإنجليزية
- **إلتزام يومي** مع تقويم بصري
- **تتبع التقدم** مع رسوم بيانية وتقييم أداء
- **شات فوري** بين المدرب والمتدربين
- **لوحة إدارة** متكاملة للمدربين والإدارة

## الشاشات والوظائف

### 🔐 المصادقة
| الشاشة | الوظيفة |
|--------|---------|
| LoginScreen | تسجيل الدخول + دعم Guest |
| RegisterScreen | إنشاء حساب جديد |
| ForgotPasswordScreen | إعادة تعيين كلمة المرور بالكود |
| PendingScreen | انتظار موافقة المدرب |
| SubscriptionScreen | الاشتراك برفع إيصال الدفع |
| FirstRunWizardScreen | شاشة ترحيب عند أول تشغيل |

### 🏠 الرئيسية
- إحصاءات: إجمالي الجلسات، سلسلة الأيام، حضور الشهر
- جلسة اليوم مع زر بدء التمرين
- وصول سريع لجميع الأقسام
- أحدث الأرقام القياسية
- تنبيه انتهاء الاشتراك

### 💪 التمرين
- **WorkoutScreen**: قائمة جلسات البرنامج المعين
- **SessionScreen**: جلسة تفاعلية كاملة تشمل:
  - بروتوكول الإحماء
  - مجاميع تفاعلية مع +/- للوزن والعدات
  - قيم الجلسة السابقة
  - حساب 1RM (Epley) والحجم
  - مؤقت الراحة مع صوت
  - بدائل التمارين
  - حفظ تلقائي + مزامنة
  - اكتشاف أرقام قياسية جديدة

### 🍽️ التغذية
- حلقة السعرات اليومية
- شريط الماكروز (بروتين/كارب/دهون/ألياف)
- متتبع الماء (بالكؤوس)
- قوائم الوجبات (فطور/غداء/عشاء/خفيفة)
- **FoodSearchScreen**: بحث في 60+ صنف غذائي

### 📅 الإلتزام
- تقويم شهري بصري
- تسجيل: حضور/غياب/راحة لكل يوم
- إحصاءات نسبة الإلتزام
- تنقل بين الأشهر

### 📈 التقدم
- تقييم أداء ذكي (ممتاز/جيد/ثبات/إنخفاض)
- رسم بياني للحجم الأسبوعي (Bar Chart)
- رسم بياني لتتبع الوزن (Line Chart)
- جدول الأرقام القياسية
- شاشة القياسات الجسدية
- شاشة صور التقدم

### 💬 الشات
- Polling كل 8 ثواني
- دعم رسائل المدرب بشارة مميزة
- تثبيت الرسائل
- حذف الرسائل (للمشرفين)
- حظر/كتم المستخدمين (للمشرفين)

### ⚙️ الإعدادات
- 4 ثيمات: داكن/فاتح/فاخر/رياضي
- 9 ألوان تمييز قابلة للتخصيص
- دعم عربي/إنجليزي + RTL/LTR
- تغيير كلمة المرور والاسم
- مؤقت الراحة (60-300 ثانية)
- تبديل تشغيل/إيقاف للمزايا
- إعداد وتجربة الاتصال بالسيرفر

### 🛡️ لوحة الإدارة (Admin/SuperAdmin فقط)
- قائمة المستخدمين مع بحث وفلترة
- موافقة/رفض الحسابات الجديدة
- تعيين البرامج التدريبية والأيام
- تعديل الماكروز اليومية لكل مستخدم
- إدارة الأدوار (SuperAdmin فقط)
- طلبات الاشتراك (موافقة/رفض + تفعيل تلقائي)
- إدارة الحظر
- أكواد الخصم (إنشاء/حذف)
- سجل التعديلات (Audit Log)

## البيانات والمزامنة

### مصدر البيانات
- **السيرفر أولاً**: كل البيانات تُحفظ في Google Apps Script (Google Sheets)
- **SQLite كاش فقط**: لا يُستخدم كمصدر أساسي

### آلية المزامنة
```
تعديل محلي
    ↓
حفظ في SQLite (فوري)
    ↓
إضافة إلى Sync Queue
    ↓
إرسال للسيرفر (إذا online)
    ↓ إذا offline:
إعادة محاولة تلقائية عند الاتصال
(Auto-sync كل 30 ثانية + عند تغيير الشبكة)
```

### معالجة الأخطاء
- حتى 5 محاولات إعادة لكل عنصر في Queue
- بعد 5 فشل: حذف من Queue (تجنب التراكم)
- عرض شريط "غير متصل" في أعلى الشاشة
